Tentang Saya

Halo! Saya adalah mahasiswa Prodi DKV di ISBI Tanah Papua. Saya tertarik dengan desain grafis dan media interaktif.

Di mata kuliah ini, saya akan belajar membuat website interaktif menggunakan HTML, CSS, dan JavaScript.

“Design is not just what it looks like and feels like. Design is how it works.” — Steve Jobs

Tag HTML yang Sudah Saya Pelajari

Tag pertama yang saya pelajari adalah <h1> untuk judul besar, dan <p> untuk paragraf. Sekarang saya juga sudah belajar <ul>, <a>, dan tag semantik seperti <header>, <main>, dan <footer>.

Kurikulum Mata Kuliah Ini

  • HTML — Struktur Web
    • Tag dasar: heading, paragraf, link
    • Gambar dan media
    • Struktur semantik
  • CSS — Tampilan Web
    • Warna, font, dan spacing
    • Box Model dan layout
  • JavaScript — Interaktivitas
    1. Variabel dan fungsi dasar
    2. Memanipulasi DOM
    3. Event listener

Langkah Belajar HTML Saya

  1. Membuat file HTML pertama (Hello World!)
  2. Memahami struktur DOCTYPE, head, dan body
  3. Belajar elemen teks: heading, paragraf, strong, em
  4. Membuat link, gambar, dan list
  5. Membangun halaman profil semantik lengkap
  6. Menambahkan CSS untuk tampilan visual
